Introduction

Python is a widely-used programming language known for its readability and simplicity. It was created by Guido van Rossum and first released in 1991.

Python emphasizes code readability with its clear syntax, which allows developers to express complex ideas in fewer lines of code.

Python is a high-level language, meaning that it abstracts much of the complexity involved in lower-level languages like C or C++.

One of Python’s biggest advantages is its large standard library, which provides many useful modules and functions for tasks like file I/O, data manipulation, and networking.

Python is platform-independent, meaning it can run on various operating systems such as Windows, Mac OS, and Linux.

# Python is easy to learn 
                    print("Hello, World!")

This introductory program outputs the text "Hello, World!" to the console. It demonstrates Python’s simplicity and ease of use.

Python is open-source, meaning anyone can contribute to its development or use it without licensing fees, which has greatly contributed to its popularity.

Python supports both imperative and object-oriented programming, making it a flexible choice for different types of projects.

Python Basics

In Python, you can define variables without explicitly declaring their type. The type of the variable is inferred at runtime based on the value assigned to it.

For example, assigning `x = 10` makes `x` an integer, while assigning `x = "hello"` would change `x` to a string.

# Variables and arithmetic operations
                    x = 10
                    y = 5
                    sum = x + y
                    print(sum)  # Output: 15

Python uses dynamic typing, meaning variables can change types dynamically as the program runs. This makes Python flexible but requires careful handling of data types.

Indentation is crucial in Python. Instead of using braces or keywords to define blocks of code, Python uses indentation levels to indicate the hierarchy of code blocks.

  • Variables: Used to store data values
  • Data types: Specify the type of value stored in a variable
  • Basic operators: Such as `+`, `-`, `*`, and `/` for arithmetic

Python uses whitespace to organize code, making it essential to follow proper indentation rules. Incorrect indentation will result in errors.

Comments in Python begin with a `#` symbol and are ignored during execution. They are used to add explanations or notes to the code.

Python Data Types

Python provides several built-in data types for handling different kinds of data. Each data type in Python is an object, meaning it has its own methods for interacting with the data.

# Example of different data types
                    num = 42  # Integer
                    pi = 3.14  # Float
                    name = "Python"  # String
                    isThisColorRed = True #Boolean - it can be either True or False

Python’s common data types include:

  • Integers: Whole numbers, such as `42` or `-7`
  • Floats: Decimal numbers, such as `3.14` or `-0.5`
  • Strings: A sequence of characters enclosed in quotes, such as `"Python"`
  • Lists: Ordered collections of items, such as `["apple", "banana", "cherry"]`
  • Dictionaries: Unordered collections of key-value pairs, such as `{"name": "Alice", "age": 25}`

Strings in Python can be manipulated using a wide range of methods, such as `.upper()`, `.lower()`, and `.replace()`.

Lists are mutable, meaning their contents can be changed after creation. You can add, remove, or modify items within a list.

Dictionaries allow for the storage of data as key-value pairs, making them ideal for scenarios where data needs to be easily accessed using a key.

Python Control Structures

Control structures allow you to dictate the flow of a program by executing different blocks of code based on certain conditions or repeating actions multiple times.

The `if-else` statement is used for conditional execution. The code within the `if` block is executed only if the condition evaluates to `True`,
otherwise the code in the `else` block is executed.

# Conditional structure
                    if num > 5:
                        print("Greater than 5")
                    else:
                        print("5 or less")

Loops allow you to repeat a block of code multiple times. The `for` loop in Python is used to iterate over a sequence of values (like a list or a range of numbers).

Python’s `while` loop executes a block of code as long as the condition remains `True`. Be careful to avoid infinite loops by ensuring
the condition will eventually become `False`.

Nested control structures are allowed, meaning you can place loops inside conditionals or vice versa, depending on the needs of your program.

Python Functions

Functions are reusable blocks of code that can be called to perform specific tasks. In Python, functions are defined using the `def` keyword, followed by the function name and parameters.

Functions help you organize your code and avoid repetition. Once a function is defined, it can be called anywhere in the program by using its name followed by parentheses.

# Function definition and use
                    def greet(name):
                        return f"Hello, {name}!"

                    message = greet("Alice")
                    print(message)  # Output: Hello, Alice!

Functions can return values using the `return` statement. If no `return` statement is used, the function will return `None` by default.

Functions can take multiple arguments, including optional arguments with default values. This allows for more flexible function calls.

Python also supports anonymous functions, known as lambda functions, which can be useful for short, simple tasks.
